## тесты Selenium 

### как запустить
1. Установить зависимости:
```bash
pip install -r requirements.txt
```
2. Вписать в `.env`:
- `BASE_URL` (адрес Django-приложения)
- `UI_USERNAME`, `UI_PASSWORD` (админ-учетка)
- `RUN_CREDS_TESTS=1` (включить тесты с авторизацией)
- `HEADLESS=1` (или `0`, чтобы видеть браузер)
3. Запуск:
```bash
python -m pytest -q
```
3.1. Запуск отдельных UI тестов: 
```bash
python -m pytest -q ui/
```
3.2. Запуск отдельных API тестов: 
```bash
python -m pytest -q api/
```
### Какие тесты реализованы
- `test_auth.py` — вход, ошибки валидации входа, регистрация, проверка несовпадающих паролей.
- `test_dashboard.py` — загрузка дашборда и смена статуса задачи.
- `test_tasks.py` — создание задачи и валидация обязательных полей формы.
- `test_employees.py` — поиск, создание, валидация формы, редактирование и удаление сотрудника.
- `test_reports.py` — фильтрация отчета, диаграмма по сотруднику, скачивание Excel.
- `test_admin_departments.py` — создание/редактирование/удаление подразделений.
- `test_admin_users.py` — создание пользователей и валидация обязательных полей.
